# IconCan

**IconCan**, Pisi Linux projeleri için merkezi bir ikon ve yerelleştirme (i18n) kütüphanesidir. Uygulamalarınızın arayüzlerini yerelleştirmek ve görsel standartları korumak için tasarlanmıştır.

## 📂 İçerik

* **`icons/`**: Uygulama arayüzlerinde kullanılan ikonlar ve semboller koleksiyonu.
* **`lang-all/`**: Desteklenen diller için ülke bayrakları ve yerelleştirme (i18n) varlıkları.

## 🌍 Yerelleştirme ve Bayraklar
`lang-all/` klasörü, uygulama içerisindeki dil seçeneklerini görselleştirmek için ihtiyaç duyulan yüksek kaliteli ülke bayraklarını içerir. 

* **Kullanım:** Uygulamanızdaki dil seçim menülerinde (dropdown) bu bayrakları kullanarak kullanıcı deneyimini zenginleştirebilirsiniz.

## 🚀 Hızlı Başlangıç

Bu depoyu projelerinize dahil etmek için:

```bash
git clone [https://github.com/pisilinux-3/iconcan.git](https://github.com/pisilinux-3/iconcan.git)
```
Ardından, ihtiyacınız olan ikon veya bayrak dosyasını ilgili yoldan (/icons veya /lang-all) projenizin assets dizinine kopyalayabilirsiniz.
